О проекте


Данный веб-сайт является демонстрационной версией решения для автоматизации Нумерологических расчётов и составления описания Матрицы Судьбы на основе результатов вычислений.

Ограничения демонстрационной версии:
- База данных ограничена одним разделом
- Оригинальный контент в базе данных заменен 'текстовыми вставками'

Стек технологий:
- Python
- Famework: Flask
- База данных: SQLite
- HTML/CSS: Bootstrap (измененный)
